# TypeScript / React / Next.js / React Native

When editing TS/TSX, follow the vendored conventions in
`~/.claude/references/typescript/` (start at its `README.md`). For React/Next
architecture the base is **bulletproof-react**; the README also links the
authorities (react.dev, Next.js docs, React Native docs, TypeScript Handbook,
Total TypeScript).

Read the relevant guide before substantial work, and match the existing
project's style where it differs.
